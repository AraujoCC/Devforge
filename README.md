# 🚀 DevForge

Fullstack social platform for developers.

## 🧠 Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- Flyway
- WebSocket

### Frontend
- React
- Vite
- TypeScript
- TailwindCSS

---

## 🔐 Features

- Authentication with JWT
- Real-time chat (WebSocket)
- Posts and projects system
- Scalable architecture (Clean Architecture)

---
